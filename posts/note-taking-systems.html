I Tried Every Note-Taking System. Here's What Stuck

May 4, 2026 · 6 min read

At various points I've used a Zettelkasten with numbered index cards, a bullet journal with its own symbol legend, and three different note-taking apps, each promising to be the last one I'd ever need. I want to say one of them was the answer. None of them were, and the reason is dumber than any of the systems themselves.

The problem was never the system

Every one of these methods works, in the sense that if you follow the instructions, you end up with organized, retrievable notes. What none of them solved was the actual bottleneck, which was that I wouldn't write a note down in the moment I had the thought, because the "proper" system was never open, or required too many steps to file something correctly.

A perfect filing system for notes you never captured is not a note-taking system. It's a very organized empty room.

What actually works now

The system that finally stuck has almost no structure. One plain text file, added to at the bottom, timestamped, no folders, no tags, no linking. When I have a thought worth keeping, I open the file and type it, and that's the entire process — capture first, organize never, unless I'm specifically looking for something later.

Once a month or so I skim back through and pull anything worth developing into an actual draft. Everything else just sits there, which is fine — an unused note costs nothing to keep around, but a system so demanding that I stop using it costs everything.

The lesson underneath all of it

Every elaborate system I tried was solving a retrieval problem I didn't really have yet, at the cost of a capture problem I had constantly. If you're choosing between a note-taking method that's elegant to browse and one that's effortless to add to in the moment, the second one wins every time, because notes you never write down don't need a system at all.
